# ORBI Atlassian Artifact Prompt

Use this when preparing Jira/Confluence-ready artifacts for ORBI.

Rules:

- Inspect before changing.
- Summarize what exists.
- Propose exact changes.
- Ask for approval before creating/updating significant Jira or Confluence artifacts.
- Do not delete anything.
- Do not rename anything without approval.
- Do not touch Aegis or CiteVyn.
- Keep all changes Free-tier safe.

When creating Jira-ready issue text, include:

```text
Product:
stableKey:
riskTier:
Workstream:
Release Target:
AI Capability:
```

When creating Confluence-ready pages, include:

```text
Template Name:
Version:
Owner:
Last Updated:
Used For:
Mandatory / Optional:
Related Jira Issue Type:
```
