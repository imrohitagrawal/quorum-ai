# Generic Study and Publishing Pack

This optional pack contains generic guidance for converting project evidence into Git-backed study artifacts and Confluence-ready pages.

It is not Aegis-specific. It must be driven by the current product's approved requirements, architecture, tests, release evidence, and source-of-truth logs.

## Use with

- `study-artifact-publisher`
- `project-knowledge-base-publisher`
- `faq-wiki-generator`
- `technical-article-writer`
- `linkedin-technical-post-writer`
- `diagram-media-standards-governor`
- `git-confluence-publish-reviewer`

## Rule

Do not publish externally without explicit human confirmation and post-write verification.
