# ORBI Optional Profile

This profile contains the reviewed ORBI / Orbisynth-AI operating model pack.

It is intentionally inactive by default so general enterprise products are not forced to follow ORBI-specific rules.

Activate only when the product is meant to run under ORBI:

```bash
make apply-orbi-profile
```

Then ask Codex to read `AGENTS.md` and `AGENTS.ORBI.md`.
