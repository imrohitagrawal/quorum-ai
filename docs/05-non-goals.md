# Non-Goals

## MVP Non-Goals

- Do not write implementation code before factory lifecycle gates are complete.
- Do not guarantee factual correctness or present the synthesis as certified truth.
- Do not make medical, legal, financial, safety, or regulated decisions for the user.
- Do not allow sensitive/private data submission as a supported safe use case before privacy controls, retention rules, logging policy, and provider-processing behavior are defined.
- Do not build multi-user team administration in the first slice.
- Do not build billing, subscriptions, quotas, or payment flows in the first slice unless cost controls later make them mandatory.
- Do not build a full research workspace, project folders, or long-term memory in the first slice.
- Do not publish Jira, Confluence, public articles, or social posts without explicit human confirmation.

## Deferred Scope

- Account system and user history.
- Enterprise admin controls.
- BYO provider keys.
- Advanced source-quality scoring.
- Evaluation datasets and automated hallucination benchmarks.
- Export/share workflows.
- Organization-level audit logging.
- Production privacy controls for sensitive data.
